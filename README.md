## Feature
- Google/Facebook Auth
- Post an Image/Video
- Display Posts
- Search Posts
- Comment to a post
- Like to a post
- Delete a post


## Tech:
- Flutter/Dart
- Riverpod2 : State management framework
- Firebase (Firestore, Storage) : Realtime DB, Media Storage
